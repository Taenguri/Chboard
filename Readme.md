# ERD

![ERD.png](/img/ERD.png)<br/><br/>

## users
user_no = 회원 번호<br/>
user_id = 회원 아이디<br/>
user_name = 회원 이름<br/>
user_password = 회원 비밀번호<br/>
verify = 회원 구분 ( Ex : 관리자, 사용자 )

## board
board_no = 글 번호<br/>
title = 제목<br/>
content = 내용<br/>
writer = 글 작성자<br/>
regdate = 글 작성일<br/>
hit = 조회수<br/>
new_mark = 뱃지 여부 ( Ex : 최신글 new뱃지 붙히기)<br/><br/><br/>
